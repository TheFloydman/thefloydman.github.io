Welcome to the Myst (2021) Save Editor, a brower-based program that can edit save files (.sav) for the remake of the video game Myst released in 2021. I have created a few other game-related resources that you can view here.

To begin, click the Choose File button, and locate the save file you would like to edit. Once you open that file, this page will automatically fill with variables to play with. When you are finished, press Save, and a new file will be saved to your download folder.

This editor does not manipulate save files directly. Instead, when you click Save, a new file is downloaded to your download folder. You will need to copy that file into the appropriate save folder. I highly recommend you keep a backup of any save file you plan to replace, just in case.

If there is a variable you would like to edit that is not displayed here, please contact me on reddit.
