Thank you for visiting! Here are a few online projects I'd like to be public for various reasons. You can find me on Discord as Floydman#7171.
Myst (2020) Save Editor
Savegame editor programmed in Javascript to edit saves from the 2020 remake of Myst. Only tested on Windows.
Obduction Save Editor
A savegame editor programmed in Javascript to edit Obduction savegames.
Villein Number Converter
The second iteration of a number converter that mimics the Villein control panels in Obduction.
Template:VilleinNumber
Out-of-commision template for the Obduction Wiki that displays decimal numbers on the Villein control panels from Obduction. Broke during the move to Fandom.
Obduction Maze Solver
An old and poorly-written app that provides step-by-step instructions on how to solve the maze in Maray, regardless of how it is currently set up. Also for use at the Obduction Wiki. (May be broken.)
