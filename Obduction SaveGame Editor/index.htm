Welcome to the Obduction SaveGame Editor, a brower-based program that can edit savegame files (.sav) for the video game Obduction. I have created a few other Obduction resources that you can view here.

To begin, click the Choose File button, and locate the save file you would like to edit. Once you open that file, this page will automatically fill with variables to play with. Once you are finished, press Save at the bottom of the page, and a new GameState.sav will be saved to your default download folder.

As a fail-safe, this editor does not manipulate save files directly. Instead, when you click Save, a new .sav file is downloaded to your default download folder. You will need to copy the resulting GameState.sav into the appropriate save folder. I highly recommend you make a backup of any save file you plan to replace. (Keep in mind, though, that using this editor will never mess up your existing save file. That can only happen if you delete or overwrite your existing save file.)

If there is a variable you would like to edit that is not displayed here, please contact me on reddit.
